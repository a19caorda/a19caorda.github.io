La lista de atributos admitidos para la etiqueta < area> son:

shape = Indica la forma del área del mapa

coords = Indica las coordenadas(x,y) del área del mapa

href = Dirección del recurso vinculado.

nohref = Indica que el área no es activa, no tiene enlace.

alt = Describe el área o su vínculo, es necesario por que area no tiene contenido.

accesskey = Define una tecla de acceso rápido. Importante para la accesibilidad.

tabindex = Asigna un número de posición en el orden de tabulación.
