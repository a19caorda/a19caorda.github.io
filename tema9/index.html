Instalación Odoo

Image 1. Vamos a la página de descarga de Odoo, y pulsamos en la última version estable, en este caso la 13

Image 2. En esta página vamos a la mejor opción para nuestro sistema operativo, en mi caso la carpeta deb

Image 3. Cuando entremos buscamo lo último de lo último, es decir, donde ponga latest

Image 4. Cuando se haya descargado, clicamos el archivo, en ubuntu nos avisará de que es un archivo externo que tengamos cuidado. Le damos a run

Image 5. Se abrirá esta ventana y clicamos Install

Image 6. Esperamos a que termine...

Image 7. Cuando haya terminado nos metemos en el navegador y poner la siguiente URL: localhost:8069

Image 8. Al ser la primera que entramos nos pedirá que creamos la base de datos, introduciendo nuestros datos. (Si pulsamos la caja de Demo data nos cargará datos de demostracion)

Image 9. Una vez aquí, podremos ya instalar aplicaciones de Odoo, como demostración instalaré la app de Inventario

Image 10. Cuando se configure saldrá una pantalla parecida a esta. Vamos a crear un producto, para ello vaya a Datos Principales > Productos

Image 11. Ahora estamos viendo los productos, le damos a Crear

Image 12. Introducimos los datos del producto a crear y pulsamos Guardar.

Image 13. Ya estará creado, ahora vamos a cambiar el número de unidades.

Image 14. Nos metemos en el producto, y pulsamos en Actualizar la unidad

Image 15. Y ahora en crear y nos saldrá para escribir la cantidad

Image 16. Pulsamos Guardar y se guardará la nueva cantidad del producto
